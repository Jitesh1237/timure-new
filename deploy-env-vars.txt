============================================
  HOW TO SET ENV VARS ON RENDER
============================================

1. Go to https://dashboard.render.com
2. Click your web service
3. Click "Environment" in left sidebar
4. Click "Add Environment Variable"
5. Paste Key and Value from above
6. Click "Save"

============================================
  HOW TO SET ENV VARS ON VERCEL
============================================

1. Go to https://vercel.com/dashboard
2. Click your project
3. Click "Settings" tab
4. Click "Environment Variables" in left sidebar
5. Add Key and Value from above
6. Click "Save"

============================================
  AFTER BOTH ARE DEPLOYED
============================================

1. Get your Render URL from the dashboard (e.g. https://timure-yatayat-api.onrender.com)
2. Go to Vercel → Settings → Environment Variables
3. Update VITE_API_URL with your actual Render URL
4. Redeploy Vercel project
